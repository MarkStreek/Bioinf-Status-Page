About This Project

This web application was created on behalf of the bioinformatics department of the Hanze University of Applied Sciences.
The purpose of this application is to provide a simple overview of the status of the workstations.
Additionally, we created a function that shows the map for every room.
With this map, you can see where the workstations are located in the room.
Github Page of the project

The status function

Inside a config file, the information about the workstations is stored.
This information is used to create the workstation cards.
In addition, every workstation retrieves a modal with more information about the workstation.
The modal can be shown by pressing a button.

The status of the workstations is retrieved by a request to the back-end.
The data from the request comes in sorted by the following properties:

  1. Status
  2. Load
  3. Memory
  4. Temperature

The first four workstations are then selected and used as a suggestion for SSH.
Additionally, the data is used to automatically update the workstations every 2 seconds.

The map function

For the map function, the same config file is being used. For every workstation, name and location are defined.
The locations are made up of matrices.
Therefore, this makes it easy to make adjustments when a workstation is moved.

When pressing a button, the information of the config file is retrieved for the given room.
For every workstation, a div is being created and for empty spaces in the matrix, an empty div is being created.
The main pc of every room (teacher pc) is at the top of the map.
After creating the map, a request is made to the back-end for information.
With this information, the divs are updated.
Please note, the data on this page is not automatically updated.
